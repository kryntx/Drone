[![Platform](https://img.shields.io/badge/platform-STM32-blue)](https://www.st.com/)
[![Toolchain](https://img.shields.io/badge/toolchain-Keil%20MDK-green)](https://www.keil.com/)

## 项目简介
本项目是**微型四轴无人机飞控固件**，基于**STM32F401CCU6**主控芯片开发，采用STM32CubeMX自动生成外设初始化代码，在Keil MDK开发环境完成编译与调试。固件计划实现**姿态解算、电机驱动、无线遥控、气压定高、光流悬停、超声波测距、卫星定位、片内Flash断电数据保存**等核心飞控功能，适配空心杯电机小四轴，支持室内外稳定飞行。

## 硬件平台
| 组件 | 型号 | 详细说明 |
| ---- | ---- | ---- |
| 主控芯片 | STM32F401CCU6 | Cortex-M4内核，主频84MHz，内置256KB Flash，满足飞控实时运算、存储及断电保存需求 |
| 姿态传感器 | MPU6050 | 6轴惯性传感器（3轴加速度+3轴陀螺仪），采集飞行姿态原始数据 |
| 气压传感器 | SPL06-001 | 高精度气压传感器，用于无人机高度解算与定高控制 |
| 光流传感器 | GL-9306 | 视觉光流模块，实现水平位置修正与定点悬停 |
| 无线通信 | NRF24L01+ | 2.4G无线收发模块，与遥控器建立双向通信，传输控制指令 |
| 卫星定位 | AIR530 | 北斗/GPS双模定位模块，默认波特率9600，支持室外定位悬停 |
| 超声波测距 | HC-SR04 | 近距离测距模块，辅助低空定高，提升低空飞行稳定性 |
| 电机驱动 | 集成MOS管（4路） | 驱动小四轴空心杯电机，输出PWM调节转速 |
| 供电系统 | 1S锂电池（3.7V-4.2V） | 整机供电，稳压至3.3V为STM32主控及传感器供电 |

## 软件架构
- **运行框架**：裸机开发，主循环轮询 + 定时器中断调度，保证飞控实时性
- **硬件驱动层**
  - I2C通信：驱动MPU6050姿态传感器、SPL06-001气压传感器
  - SPI通信：驱动NRF24L01+无线模块
  - UART通信：驱动GL-9306光流模块、AIR530定位模块
  - 定时器：驱动HC-SR04超声波模块触发与测距、电机PWM输出
- **核心算法层**
  - 姿态解算：Mahony互补滤波 + 四元数解算，输出稳定姿态角
  - 飞行控制：双环PID控制（角度环→角速度环），输出电机PWM控制量
  - 数据融合：气压计+超声波+光流数据融合，优化定高与悬停精度
- **应用功能层**
  - 遥控解析：自定义NRF24L01数据包，解析油门、横滚、俯仰、偏航控制量
  - **断电存储**：基于STM32片内Flash实现参数掉电保存，无需外部EEPROM

## 核心功能详解
### 1. 片内Flash断电数据保存（核心特色功能）
本项目**无需外接EEPROM**，直接利用STM32F401CCU6片内Flash实现参数掉电非易失性存储，上电自动加载，断电自动保存。
- **存储位置**：占用STM32F401最后一个空白Flash扇区，不影响程序运行
- **保存数据**
  1. PID参数（角度环、角速度环、定高环PID）
  2. 传感器校准参数（陀螺仪零偏、加速度计校准值）
  3. 遥控器校准数据（摇杆中点、行程范围）
  4. 飞控配置参数（电机顺序、PWM频率、传感器量程）
- **工作逻辑**：上电自动读取Flash参数→校准完成后自动写入保存→参数修改后一键写入Flash

### 2. 全功能飞控控制
- 姿态稳定：六轴传感器+互补滤波/四元数，实现自稳飞行
- 多级定高：气压计+超声波双数据源，低空高精度定高
- 定点悬停：光流模块水平修正，配合定高实现室内悬停
- 无线遥控：NRF24L01 2.4G双向通信，抗干扰强
- 安全保护：电机一键解锁/上锁，防止误触启动

## 编译与烧录
### 开发环境要求
- IDE开发工具：Keil MDK v5.31 及以上版本
- 编译工具链：ARM Compiler v6
- 芯片支持包：Keil.STM32F4xx_DFP.2.x.x
- 下载调试器：ST-Link / J-Link / DAP-Link

### 操作步骤
1. **克隆项目源码**
```bash
git clone https://github.com/kryntx/Drone.git
cd Drone
```
2. **打开Keil工程**
进入`MDK-ARM/`目录，双击`Drone.uvprojx`工程文件
3. **编译工程**
按下快捷键`F7`或点击Keil界面`Build`按钮，完成代码编译
4. **固件烧录**
将调试器连接飞控板SWD接口，按下快捷键`F8`或点击`Download`按钮，完成固件下载

## 飞行使用说明
1. **上电初始化**：飞控通电后，自动加载Flash存储参数→传感器自检→LED闪烁表示初始化完成
2. **遥控对频**：打开NRF24L01遥控器，等待接收机指示灯常亮，对频成功
3. **电机解锁**：遥控器油门拉至最低 + 方向摇杆最右，电机进入怠速状态
4. **操控飞行**：缓慢推油门起飞，通过摇杆控制横滚、俯仰、偏航；低空自动启用超声波定高
5. **参数保存**：校准/调参完成后，自动写入Flash，断电不丢失
6. **电机锁定**：遥控器油门拉至最低 + 方向摇杆最左，电机停止运转
