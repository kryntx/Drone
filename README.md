# Drone - 基于 STM32F401CCU6 的小四轴无人机
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-STM32-blue)](https://www.st.com/)
[![Toolchain](https://img.shields.io/badge/toolchain-Keil%20MDK-green)](https://www.keil.com/)

## 简介
本项目是一款微型四轴无人机飞控固件，主控采用STM32F401CCU6，依托STM32CubeMX完成外设初始化配置，在Keil MDK开发环境下编写调试。固件集成姿态解算、电机调速、2.4G无线遥控、气压定高、光流定点悬停等核心飞行功能。

## 硬件平台
| 组件 | 型号 | 功能说明 |
| ---- | ---- | ---- |
| 主控芯片 | STM32F401CCU6 | Cortex-M4内核，84MHz主频，256KB闪存 |
| 姿态惯性单元 | MPU6050 | 六轴加速度+陀螺仪，采集飞行姿态数据 |
| 高精度气压计 | SPL06-001 | 气压高度采集，实现自主定高飞行 |
| 光流定位模块 | GL9306 | 水平位移检测，辅助定点悬停 |
| 无线遥控模块 | NRF24L01+ | 2.4G无线通信，接收遥控器飞行指令 |
| 动力驱动 | 四路集成MOS管 | 驱动空心杯高速电机运转 |
| 供电电源 | 1S 3.7V锂电池 | 整机供电，升压5V为传感器供电 |

## 软件架构
- 运行框架：裸机开发，主循环轮询+定时器中断调度
- 通信驱动：I2C驱动姿态/气压传感器、SPI驱动无线模块、串口驱动光流模块
- 姿态算法：Mahony互补滤波+四元数姿态解算
- 飞行控制：双环PID控制（角速度环+角度环），输出PWM电机控制量
- 遥控系统：自定义NRF24L01通信协议，解析油门、横滚、俯仰、偏航摇杆数据
- 融合算法：气压高度与光流数据融合，优化悬停稳定性

## 编译与烧录
### 开发环境配置
- 编译软件：Keil MDK5 及以上版本
- 编译内核：ARM Compiler Version 6
- 芯片固件库：STM32F4系列器件支持包
- 下载调试工具：ST-Link、J-Link、DAPLink

### 部署流程
```bash
# 拉取项目源码
git clone https://github.com/kryntx/Drone.git
cd Drone
```
1. 进入`MDK-ARM`文件夹，打开`Drone.uvprojx`工程
2. 按下`F7`一键编译整个项目代码
3. 调试器连接飞控SWD下载口，按下`F8`完成固件烧录

### 硬件引脚分配
| 外设设备 | 硬件引脚 | 通信方式 |
| ---- | ---- | ---- |
| MPU6050 | PB8(SCL)、PB9(SDA) | I2C1 |
| SPL06-001 | PB8(SCL)、PB9(SDA) | 共用I2C1总线 |
| NRF24L01 | PA4、PA5、PA6、PA7 | SPI1 |
| GL9306光流 | PA2(TX)、PA3(RX) | USART2 |
| 一号电机 | PA0(TIM2_CH1) | PWM调速 |
| 二号电机 | PA1(TIM2_CH2) | PWM调速 |
| 三号电机 | PB0(TIM3_CH3) | PWM调速 |
| 四号电机 | PB1(TIM3_CH4) | PWM调速 |

## 飞行使用教程
1. 上电初始化：设备通电后指示灯闪烁，代表所有传感器初始化完成
2. 遥控器对频：开启2.4G遥控器，等待飞控接收模块指示灯常亮即配对成功
3. 电机解锁：遥控器摇杆油门拉至最低，方向摇杆拨至最右侧，电机进入怠速状态
4. 操控飞行：缓慢推动油门摇杆升空，其余摇杆控制机身前后左右、转向飞行
5. 上锁停机：油门归最低，方向摇杆拨至最左侧，电机停止运转

## 项目优化方向
1. 优化陀螺仪零点校准算法，降低飞行姿态漂移
2. 完善气压计温度补偿程序，解决高空定高偏移问题
3. 适配优化GL9306光流模块数据解析协议
4. 统一收发端无线数据包格式，提升遥控通信稳定性
5. 新增飞行模式切换、低电量报警等实用功能
